# SignalForge

SignalForge — персональная система для преобразования потока сообщений и материалов в структурированную базу знаний.

Первый планируемый сценарий — анализ Telegram-чата об LLM, AI-native разработке и coding agents: импорт истории, обработка новых сообщений, извлечение полезных ссылок и подготовка кратких дайджестов.

## Current phase

Проект находится на этапе подготовки Spec-Driven Development инфраструктуры.

Сейчас разрешено создавать только:

- правила разработки;
- структуру спецификаций;
- шаблоны спецификаций;
- instruction-only skills для Codex.

Приложение, зависимости, база данных и Telegram-интеграция не создаются до появления соответствующих утверждённых спецификаций.

Первая feature импортирует доступную историю одного Telegram-чата через user-session в PostgreSQL. Подробная безопасная настройка, миграции и запуск описаны в [`docs/operations/telegram-history-import.md`](docs/operations/telegram-history-import.md).
